XPS Analysis

This project focuses on chemical state analysis of thin film samples using X-ray photoelectron spectroscopy.

Project Overview

XPS is used to identify chemical bonding states such as Si-N, Si-O, metal-nitride bonding, oxidation, and surface contamination.

Core Levels Studied

  • Si 2p
  • N 1s
  • O 1s
  • C 1s
  • Metal core levels
